# Анализ эффективности внедрения новой механики оплаты услуг на сайте

В рамках проекта решены следующие задачи:
1. Подсчет метрик ARPPU и конверсии в оплату в тестовой и контрольной группах.
2. Сформулированы гипотезы для проверки статистической значимости проведенного исследования.
3. Проверка метрик на нормальность, равенство дисперсий для выбора статистического критерия.
4. Применены статистический критерий Хи-квадрат для конверсии в оплату и Т-тест Стьюдента для оценки ARPU
5. По результатам тестирования сделаны выводы по гипотезам.
6. Сделаны вывода о необходимости внедрения новой механики оплаты услуг.
7. Написана функция, дополняющая основной датафрейм новыми данными.
8. Написана функция, строящая график по метрика.

Стек: pandas, numpy, matplotlib, pingouin.
